# Flashcard Learning Application

## Project Title
Flashcard Learning Application

## Overview of the Project

The Flashcard Learning Application is a simple Python program made to help students study more effectively. It lets users create digital flashcards by adding questions and answers, organize them by subjects, and practice using an interactive quiz. All the data is saved so it stays safe even after closing the program.

## Features

- Create flashcards with questions and answers  
- Organize flashcards by subject  
- View all saved flashcards easily  
- Practice using a quiz mode with random questions  
- Get instant feedback and score while practicing  
- Save data using a JSON file  

## Technologies / Tools Used

- Python 3  
- JSON for storing flashcard data  
- Visual Studio Code (VS Code)  
- GitHub for version control and project hosting
